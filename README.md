# FLIF-GUI
Front end for FLIF: free lossless image format.
## Windows installer + FLIF Windows binaries
* Drag & drop
* Batch processing
* Parallel processing
* Size statistics
***
Built with XML, using [FileMill](https://FileMill.net).

Command-line tool: [FLIF](https://github.com/FLIF-hub/FLIF).
